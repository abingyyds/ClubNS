Global Inheritance Strategy

Set your default inheritance behavior when registering a new domain. When a domain you register has been used by someone else, this setting decides whether to inherit the existing club member data.

Auto Inherit: Automatically inherit all member data and settings from the original club
Never Inherit: Do not retain any member data from the original club, start completely anew
Manual Confirm: You manually decide whether to inherit member data after registering a domain
